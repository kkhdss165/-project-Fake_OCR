# Fake_OCR
3학년 2학기 '영상처리프로그래밍' 기말 프로젝트
(**컴퓨터비전, 인공지능을 사용하지 않고** 영상처리 기술만 사용하는 프로젝트)

# <소개>
  - Python Opencv 사용
  - 사진 속의 텍스트를 추출하는 인공지능 **OCR(Optical Character Recognition)** 모티브
  - Opnecv의 이진화(binarization), 컨투어(contour), 비트연산(bitwise) 활용
